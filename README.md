# LuaTunes
- iTunes store 따라한 LuaLaTeX 예제
- https://itunes.apple.com/search?term=<keyword>'

  - keyword: 'kpop', 'mariah_carey', etc.

- TeXLive 2015
- dkjson.lua(https://luarocks.org/modules/luarocks/dkjson)
- ko.TeX
- fonts
  - NotoSansKR-Light
  - Helvetica-Light

## compile
- lualatex itunes
